# PointerFLY's Personal Website

This is the repository for my personal website and blog, built with [Jekyll](https://jekyllrb.com/).

## Local Development

Ensure you have Ruby and Bundler installed, then run:

```bash
# Install dependencies
bundle install

# Start local server with future posts and drafts
bundle exec jekyll serve --future --drafts
```
